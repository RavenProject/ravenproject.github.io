What are the Ravencoin transaction details?

This screen shows details about the transaction:

Status

When sending, the recipient will usually receive the transaction in 1-10 minutes on the regular processing speed setting, and 1-24 hours on economy. For receiving, we have no control over how long it will take for the transac tion to complete.
If the transaction status is "complete," then the transaction has successfully gone through. If this is a transaction you sent, the money is now in the wallet of the recipient. If this is a transaction you received, the money is now in your wallet, ready to be spent.

If the status is not yet complete, this will show the current progress.

Memo

Here you can attach a note to the transaction to remind yourself what it was for. This memo will not be shared with anyone else and is only visible to you, but you can edit it at any time, even after the transaction has completed. Amount

This shows detailed information about how much Ravencoin was sent, how much it was worth at the time, and other details about your balance.

Other

At the bottom of this screen you will find additional technical details about the transaction. The Ravencoin transaction ID can be used as proof that a payment was made.
